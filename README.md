# Python医药信息管理系统 python220

## 项目概述

本项目是一个基于Python的医药信息管理系统，系统支持药品、顾客和经办人信息的查询、录入、删除、修改以及数据报表和信息浏览等核心功能。

## 技术栈

- **前端技术**：HTML、CSS、JavaScript、jQuery
- **后端技术**：Python、Django
- **数据库**：MySQL
- **开发工具**：PyCharm、Navicat

## 系统运行环境

- Python 3.0及以上
- MySQL 8.0

## 功能模块

- **信息查询**：支持药品、顾客和经办人信息的多条件查询。
- **信息录入**：药品、顾客和经办人信息的录入。
- **信息删除**：支持对不再需要的信息进行删除。
- **信息修改**：对已录入的信息进行更新修改。
- **数据报表**：生成各类信息的统计报表。
- **信息浏览**：直观展示各类信息列表。

## 系统角色

- 系统管理员：负责系统的维护与信息管理。
- 普通用户：进行信息查询和浏览。

## 部署步骤

1. 确保Python环境和MySQL数据库已正确安装和配置。
2. 克隆或下载项目源码到本地。
3. 使用PyCharm或命令行工具，运行`pip install -r requirements.txt`安装所需依赖。
4. 根据配置文件`config/settings.py`修改数据库连接信息。
5. 运行`python manage.py makemigrations`和`python manage.py migrate`创建和迁移数据库。
6. 运行`python manage.py runserver`启动开发服务器。
